# UI Feature Ideas

Features to implement for a more modern portfolio look:

- **Gradient mesh animated background** — subtle animated gradient on the hero section instead of a flat color
- **Glassmorphism cards** — `backdrop-filter: blur()` with semi-transparent backgrounds for project/skill cards
- **Smooth page transitions** — Angular route animations (`@angular/animations`) for fade or slide between pages
- **Active nav indicator** — animated underline or pill that slides between nav items on navigation
- **Custom cursor** — subtle dot that follows the mouse, a dev portfolio staple
- **Timeline component** — a visual vertical timeline for experience/education on the About page (more engaging than a plain list)
- **Dark/light theme persistence** — save the user's theme preference to `localStorage` so it survives page refreshes
- **Subtle particle or dot grid background** — lightweight canvas animation in the background
